# Titanic Survival Prediction using KNN

This project predicts whether a passenger survived the Titanic disaster using the K-Nearest Neighbors (KNN) machine learning algorithm.

## Project Workflow

- Data Collection
- Exploratory Data Analysis (EDA)
- Handling Missing Values
- Data Preprocessing
- Feature Encoding
- Feature Scaling
- Train-Test Split
- Model Training using KNN
- Prediction and Evaluation

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset

Dataset used:
Titanic dataset from Kaggle

## Features Used

- Passenger Class
- Gender
- Age
- Siblings/Spouses
- Parents/Children
- Fare
- Embarked Location

## Machine Learning Algorithm

K-Nearest Neighbors (KNN)

## Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- Classification Report

## Output

The model predicts:
- 0 → Did Not Survive
- 1 → Survived
